### 🛡️ Our First Open-Source Weapon: **Buckl**

> **Buckle up. The AI’s just a passenger.**

Buckl wraps **every IDE and terminal launch** in a **kernel-enforced macOS sandbox** so tight the process literally cannot negotiate its way out.  

- Not a polite suggestion  
- Not a PATH trick  
- A proper macOS security policy  

Your workflow stays **exactly the same**.  
The blast radius if your AI goes full sci-fi villain? **Dramatically smaller.**

**Built with languages almost as immature as I am:**  
**Go • Rust • Node**
