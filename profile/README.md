---

### 🛡️ Buckl

> Buckle up. The AI’s just a passenger.

Buckl runs your IDE and terminal inside a macOS sandbox enforced at the kernel level.

Not advisory. Not optional. Not something a process can talk its way out of.

No PATH tricks  
No soft limits  
A real boundary

Your workflow stays the same.  
If something misbehaves, it stays contained.

Built with:
Go • Rust • Node

[⭐ Star Buckl](https://github.com/nilwind/buckl)
